# Preparacion del entorno local

Estoy trabajando desde mi laptop con Ubuntu Linux 26.04 LTS.

El objetivo es preparar el entorno para el proyecto.

## 1. Actualizar UV

Actualice `UV` con el siguiente comando:

```bash
curl --proto '=https' --tlsv1.2 -LsSf \
  https://releases.astral.sh/github/uv/releases/download/0.11.15/uv-installer.sh | sh

uv --version
```

Resultado:

```text
everything's installed!
uv 0.11.15 (x86_64-unknown-linux-gnu)
```

## 2. Crear el entorno virtual

Desde `/home/cszv/Documents/UCB-LIA/sis122/assignments/final_project`, ejecute:

```bash
uv init --bare
uv python install 3.14.5
uv python pin 3.14.5
uv venv
```

Resultado:

```text

 + cpython-3.14.5-linux-x86_64-gnu (python3.14)

Pinned `.python-version` to `3.14.5`

Using CPython 3.14.5

Activate with: source .venv/bin/activate
```
