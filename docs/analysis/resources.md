# Recursos

Medios, herramientas o documentos que se usan para realizar los procesos.

## Recursos Tecnológicos

- **MySQL**: Gestor de base de datos relacional principal del proyecto. Se usará para almacenar clientes, vehículos, reservas, contratos, pagos, devoluciones y mantenimiento.
- **MySQL Workbench**: Herramienta para diseñar el modelo entidad-relación, crear el modelo lógico/físico y ejecutar consultas SQL.
- **SQL**: Lenguaje usado para crear tablas, relaciones, restricciones, inserciones, consultas y reportes.
- **Visual Studio Code**: Editor para redactar la documentación y organizar los archivos del proyecto.
- **Git / GitHub**: Control de versiones para mantener el historial del proyecto y compartir avances entre los integrantes.
